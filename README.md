Crime Data Analysis

Project Overview
This project focuses on analyzing crime data using Exploratory Data Analysis (EDA) and data visualization techniques. The objective is to identify crime trends, victim demographics, crime hotspots, weapon usage patterns, and case status distributions to gain meaningful insights into crime behavior.

The project was implemented using Python, Pandas, NumPy, Matplotlib, and Seaborn.

Objectives
Understand the distribution of crime incidents.
Identify the most common crime types.
Analyze victim demographics such as age, gender, and ethnicity.
Explore crime patterns across different geographical areas.
Examine weapon usage and case outcomes.
Generate meaningful visualizations for crime trend analysis.

Key Findings
Battery - Simple Assault was the most frequently reported crime.
Adults and young adults were the most affected victim groups.
Crime incidents were reported almost equally among male and female victims.
Southeast, Southwest, Central, and 77th Street were identified as major crime hotspots.
Strong-arm offences were among the most common methods used in crimes.
Most reported cases were categorized under Investigation Continued.
Crime patterns varied across geographical areas and victim demographics.

Conclusion
The analysis successfully identified major crime trends, victim characteristics, high-crime locations, and case outcome patterns. The findings demonstrate how data analytics and visualization can be used to better understand crime behavior and support data-driven decision-making for crime prevention and public safety.

Recommendations
Increase surveillance and law enforcement presence in identified crime hotspot areas.
